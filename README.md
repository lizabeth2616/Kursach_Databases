# Система управления персоналом для небольшой компании

· Суть проекта: Создать БД для учета сотрудников, их должностей, отделов и зарплат.
· Основные таблицы:
  · Employees (Сотрудники): employee_id, first_name, last_name, email, hire_date, salary, department_id, manager_id (ссылка на самого себя, на руководителя).
  · Departments (Отделы): department_id, department_name, budget.
  · Positions (Должности): position_id, position_title.
· Задачи:
  1. DDL: Создать таблицы. Обратить внимание на внешний ключ manager_id, который ссылается на employee_id в той же таблице Employees.
  2. DML: Наполнить данными, создав иерархию (например, директор -> начальник отдела -> рядовой сотрудник).
  3. Выборки (SELECT):
     · Показать всех сотрудников указанного отдела.
     · Найти подчиненных для конкретного менеджера (использовать самообъединение SELF JOIN).
     · Посчитать общий фонд заработной платы по отделам (GROUP BY, SUM).
  4. Представления (VIEW): Создать представление, которое показывает сотрудников вместе с названием их отдела и ФИО их непосредственного руководителя.
  5. Хранимые процедуры: Процедура для повышения зарплаты всем сотрудникам отдела на определенный процент.
